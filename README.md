# 📊 Statistical Analysis & Machine Learning Notebooks

A collection of Python notebooks covering **core statistical concepts and ML algorithms**, implemented from scratch with real datasets.

## 📚 Topics Covered

| Notebook | Concept | Key Libraries |
|----------|---------|--------------|
| `hypothesis_testing_Q1-Q4.ipynb` | Chi-square, t-tests, ANOVA, p-values | `scipy`, `numpy` |
| `simple_linear_regression.ipynb` | OLS regression, R², residual analysis | `sklearn`, `matplotlib` |
| `multi_linear_regression.ipynb` | Multiple predictors, multicollinearity, VIF | `sklearn`, `statsmodels` |
| `logistic_regression.ipynb` | Binary classification, ROC-AUC, confusion matrix | `sklearn` |
| `clustering.ipynb` | K-Means, Hierarchical, DBSCAN | `sklearn` |
| `PCA.ipynb` | Dimensionality reduction, explained variance | `sklearn` |
